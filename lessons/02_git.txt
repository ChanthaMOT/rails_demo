Git (Source Code Management)

/**	Initilize Git
	git init

/** Pack everything in the current directory
	git add .

/**	List the status of the pack
	git status

/**	Commit the pack to the server
	git commit -m "Message"

/** Rollback to the last commited version
	git checkout -f
